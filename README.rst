编译
========

QEMU是多平台软件，旨在在所有现代设备上构建
Linux平台，OS-X，Win32（通过Mingw64工具链）和各种
其他 UNIX 目标。构建 QEMU 的简单步骤是：


.. code-block:: shell

  mkdir build
  cd build
  ../configure
  make

其他信息也可以通过QEMU网站在线找到：

* `<https://wiki.qemu.org/Hosts/Linux>`_
* `<https://wiki.qemu.org/Hosts/Mac>`_
* `<https://wiki.qemu.org/Hosts/W32>`_
